
  # Clase Muestra

  Repositorio creado para la clase muestra de la Universidad Ibero

  ## Prerrequisitos

  Para el desarrollo de esta clase, asumimos que los alumnos ya cuentan con las siguientes herramientas instaladas en sus equipos de cómputo:

  - **Python**: [https://www.python.org/downloads/](https://www.python.org/downloads/)
  - **Un editor de código (IDE)**: se estará usando Visual Studio Code, disponible en [https://code.visualstudio.com/download](https://code.visualstudio.com/download)
  - **Git** como controlador de versiones (opcional), disponible en [https://git-scm.com/downloads](https://git-scm.com/downloads), con conocimiento básico de:
    - `add`
    - `commit`
    - `push`
